# GitHubに貢献

1. レポジトリのフォークを作成します。
2. レポジトリにご希望の修正を行います。
3. レポジトリにコミットして、pushします。
4. PR（プルリクエスト）を作成します。「フォークからプルリクエストを作成する」を参照してください。
